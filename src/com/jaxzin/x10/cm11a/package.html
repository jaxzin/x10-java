Contains an implementation of the {@link javax.x10} API for the CM11A X10 transceiver. {@link com.jaxzin.x10.cm11a.CM11A} is the implementation of {@link javax.x10.X10Transceiver}. {@link com.jaxzin.x10.cm11a.CM11AConnection} is the implementation of {@link javax.x10.X10Connection}. {@link com.jaxzin.x10.cm11a.CM11ADriver} is the implementation of {@link javax.x10.X10Driver}.
